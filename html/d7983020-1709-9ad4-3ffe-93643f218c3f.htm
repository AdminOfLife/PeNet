Access the IMAGE_DOS_HEADER of the PE file.

Access the IMAGE_EXPORT_DIRECTORY of the PE file.
ImageImportDescriptors
Access the IMAGE_IMPORT_DESCRIPTOR array of the PE file.
ImageLoadConfigDirectory

ImageNtHeaders

ImageTlsDirectory
Access the IMAGE_TLS_DIRECTORY from the data directory.
ImpHash
The Import Hash of the binary if any imports are given else null;
ImportedFunctions
Access the imported functions as an array of parsed objects.
Is32Bit
Returns true if the PE file is x32.
Is64Bit
Returns true if the PE file is x64.
IsDLL
Returns true if the DLL flag in the File Header is set.
IsEXE
Returns true if the Executable flag in the File Header is set.
IsSigned
Returns true if the PE file is signed. It does not check if the signature is valid!
IsValidPeFile
Returns true if the Exception Dir, Export Dir, Import Dir, Resource Dir and Security Dir are valid and the MZ header is set.
MD5
The MD5 of hash sum of the binary.
PKCS7
A X509 PKCS7 signature if the PE file was digitally signed with such a signature.
RuntimeFunctions
Access the array of RUNTIME_FUNCTION from the Exception header.
SHA1
The SHA-1 hash sum of the binary.
SHA256
The SHA-256 hash sum of the binary.
WinCertificate
Access the WIN_CERTIFICATE from the Security header.
