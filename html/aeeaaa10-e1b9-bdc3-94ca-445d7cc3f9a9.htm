IMAGE_SECTION_HEADER Properties

The IMAGE_SECTION_HEADER type exposes the following members.

Properties
  NameDescription
Characteristics
Section characteristics. Can be resolved with
Name
Max. 8 byte long UTF-8 string that names the section.
NumberOfLinenumbers
The number of line-number entries for the section.
NumberOfRelocations
The number of relocations for the section. Is zero for executable images.
PhysicalAddress
The raw (file) address of the section.
PointerToLinenumbers
Pointer to the beginning of the line-numbers in the file. Zero if there are no line-numbers in the file.
PointerToRawData
Raw address of the section in the file.
PointerToRelocations
Pointer to the beginning of the relocation. If there are none, the value is zero.
SizeOfRawData
Size of the section in raw on disk. Must be a multiple of the file alignment specified in the optional header. If its less than the virtual size, the rest is filled with zeroes.
VirtualAddress
RVA of the section start in memory.
VirtualSize
Size of the section when loaded into memory. If it's bigger than the raw data size, the rest of the section is filled with zeroes.
