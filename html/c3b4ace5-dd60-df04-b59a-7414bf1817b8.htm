PeNet Library
PeNet is .Net library written to parse and edit Windows Portable Executable (PE) files. It's completly written in managed code and does not depend on PInvoke. For example, PeNet show imports and exports of PE files, extracts certificate revocation lists from binary signatures or computes different file hashs incuding the ImpHash which is popular in malware analysis.
