PeNet.PatternMatching Namespace
The PatternMatching namespace contains functions to match strings and byte patterns. This is useful if you want to match for malware signatures.
Classes

Trie
This trie consists of bytes (for binary files) and uses strings as values to identify the different patterns.
TrieT, TValue
Trie that will find strings or phrases and return values of type T for each string or phrase found.
